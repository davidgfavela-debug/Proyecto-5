# Análisis de Movilidad Urbana y Productividad Económica

## Objetivo
Analizar la relación entre congestión vehicular y productividad económica en distintas ciudades.

## Dataset
Datos de tráfico (tiempos de viaje, congestión) y datos económicos (PIB per cápita).

## Metodología
- Limpieza de datos
- Estandarización de variables
- Unión de datasets por ciudad-año
- Análisis exploratorio (EDA)
- Visualización de datos

## Hallazgos
- No siempre existe correlación directa entre congestión y PIB.
- Algunas ciudades presentan alta congestión con bajo rendimiento económico.
- Se detectaron outliers importantes.

## Recomendaciones
- Priorizar inversión en infraestructura en ciudades con alta congestión.
- Validar fuentes de datos para análisis más profundo.

## Herramientas
- Python
- Pandas
- Matplotlib / Seaborn
